* The following program has been tested on ubuntu 20.04 and ros noetic

* 2 - `opencv-contrib-python - 4.2.0.68`
* 3 - `opencv-python - 4.2.0.68`
* 4 - `pydobot - 1.3.2`
* 5 - `numpy - 1.22.2`
